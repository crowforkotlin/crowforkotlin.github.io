💻 Git Bash Zsh + Powerlevel10k (P10k) 极速搭建

一、前置环境准备

步骤 描述 链接
安装 Git Bash Windows 上的 Git 命令行工具，提供 MinGW 环境。 下载链接
安装 MSYS2 提供 pacman 包管理工具，便于安装 Linux 工具。 下载链接
配置环境变量 将 MSYS2 的 mingw64\bin 和 usr\bin 加入 Windows PATH。 确保 Git Bash 能识别 MSYS2 工具。

二、Zsh & Oh My Zsh 安装

步骤 描述 命令行/代码
安装 Zsh 通过下载压缩包 (usr, etc 文件) 解压到 GIT 根目录。 Zsh 包链接
设置默认 Shell 创建 ~/.bash_profile，设置 Git Bash 默认启动 Zsh。 完整代码见下文
安装 Oh My Zsh 下载并安装 Oh My Zsh 框架。 sh -c "$(curl -fsSL ...)"
安装 P10k 主题 克隆 Powerlevel10k 主题到 Zsh 自定义目录。 git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k
安装插件 克隆 autosuggestions 等增强插件到 Zsh 插件目录。 git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
